# Active State Model

Challenge:

- RECRUITING
- IN_PROGRESS
- COMPLETED

Meeting:

- VOTING
- CONFIRMED
- COMPLETED
- CANCELLED

Vote:

- PENDING
- APPROVED
- REJECTED
- EXPIRED
- CANCELLED

Member privilege:

- ACTIVE
- REVOKED

Expense execution session:

- HOLDING
- SUBMITTED
- CONFIRMED
- FAILED
- CANCELLED
- EXPIRED

Do not use legacy states such as PAUSED, CLOSED, SCHEDULED, or GRACE_PERIOD as active defaults.
